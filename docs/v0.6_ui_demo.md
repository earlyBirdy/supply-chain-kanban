# v0.6 One-command UI Demo

## Run
```bash
make demo-ui
```
Then open Superset:
- URL: http://localhost:8088
- Login: admin / admin

## What gets auto-created
- Database connection to demo Postgres (service name `db`)
- Datasets: `agent_cases`, `market_signals`
- Charts: simple table visualizations
- Dashboards:
  - Board Supply Chain Risk View
  - Crisis Mode Control Room

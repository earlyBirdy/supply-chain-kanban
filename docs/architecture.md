# Architecture Overview

Market Signals / Operational Data
        ↓
Constraint Detection Agent
        ↓
IBP Scenario Engine
        ↓
Agent Negotiation Layer
        ↓
Decision Scoring (Service / Cost / Risk)
        ↓
Kanban / Dashboards / Alerts
        ↓
Human Approval

Design principles:
- Product-agnostic
- Scenario-first
- Human-in-the-loop
- Auditable
